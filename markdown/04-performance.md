## Performance

### Basic networking performance

* Client node -> OSD node 25Gbit/s
* Multiple client nodes -> OSD node 72 Mbit/s
* OSD Node -> OSD Node 68 Mbit/s

### Raw device peformance
* SSDs
  * 4k random rw: 
  * 4k sequential rw:
  * 4m sequential rw: 
* NVMEs
  * 4k random rw: 
  * 4k sequential rw:
  * 4m sequential rw:
